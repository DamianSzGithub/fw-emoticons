FW Emoticons

Do you like Font Awesome Icons? You gotta love Font Wonderful Emoticons!

  • Free
  • Lightweight
  • Easy to use
  • Reach collection of font-based css emoticons.

No more nasty and heavy images. Perfect for blogs, personal sites, forums, social networks, and pretty much everything you want.

Installation

Add this line to your head section:

	    <link rel="stylesheet" type="text/css" media="all" href="https://cdn.rawgit.com/DamianSzGithub/fw-emoticons/master/fw_emoticons.css">

You can also download the repo, link just css to your page, and edit this component locally if you wish.

Usage

Main pattern to insert emoticon to your page looks like follow:

	    <ANY class="fw EMO_TYPE EMO_NAME"></ANY>

Current types:

Default (no type specified)
fw-o - Default (outlined)
fw-s - Sketchy
fw-f - Today I Feel
fw-m - Memes

Make sure to customize .fw class to your needs. The most common attributes will be:

		.somediv .fw {
    transform: scaleX(-1); /* Flip horizontally */
    text-shadow: 0 0 1px rgba(0,0,0,0.15); /* make 'em smooth! */
    margin-left: 2px; /* Auto-separates emoticons from text */
    font-size: 22px;
    color: red;
		}

Examples:


	  	

		  <span class="fw perky"></span>
		  
		

		

		  <span class="fw fw-o perky"></span>
		  
		

	    

		  <span class="fw fw-s pain"></span>

Animations:

Animations are perfect supplementation for this component. Emoticons are working fine with Animate.css framework, but you can also use out of the box ones:


	    

		  <span class="fw fw-m fuu fw-shake"></span>
		  
		

		

		  <span class="fw fw-m alone fw-spin"></span>
		  
		

		

		  <span class="fw fw-m trollface fw-rotate"></span>
		  
		

		

		  <span class="fw fw-m pfftch fw-updown"></span>
		  
		

		

		  <span class="fw fw-m derpina fw-bounce"></span>

To modify default animation behavior you can add these classes respectively:

Duration: fw-100ms, fw-200ms ... fw-2000ms
Delay: fw-d1s, fw-d2s ... fw-d10s
Iteration count: fw-rep-2, fw-rep-3 ... fw-rep-10
Direction: fw-reverse
